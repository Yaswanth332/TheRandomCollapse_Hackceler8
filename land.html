Integration Examples

// Generate an OTP
const generateOTP = async (email, length = 6) => {
  try {
    const response = await fetch('https://api.quantumotp.com/generate', {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json',
        'Authorization': 'Bearer qtp_sk_3f7a9b2c8d1e5f6a9c2b8d1e5f6a9c2b'
      },
      body: JSON.stringify({
        email: email,
        length: length
      })
    });
    
    const data = await response.json();
    return data.otp;
  } catch (error) {
    console.error('Error generating OTP:', error);
    return null;
  }
};

// Verify an OTP
const verifyOTP = async (email, otp) => {

import requests
import json

# Generate an OTP
def generate_otp(email, length=6):
    url = "https://api.quantumotp.com/generate"
    headers = {
        "Content-Type": "application/json",
        "Authorization": "Bearer qtp_sk_3f7a9b2c8d1e5f6a9c2b8d1e5f6a9c2b"
    }
    payload = {
        "email": email,
        "length": length
    }
    
    try:
        response = requests.post(url, headers=headers, data=json.dumps(payload))
        data = response.json()
        return data.get("otp")
    except Exception as e:
        print(f"Error generating OTP: {e}")
        return None

# Verify an OTP
def verify_otp(email, otp):
    url = "https://api.quantumotp.com/verify"
    headers = {
        "Content-Type": "application/json",
        "Authorization": "Bearer qtp_sk_3f7a9b2c8d1e5f6a9c2b8d1e5f6a9c2b"
    }
    payload = {
        "email": email,
        "otp": otp
    }
    
    try:
        response = requests.post(url, headers=headers, data=json.dumps(payload))
        data = response.json()
        return data.get("valid", False)
    except Exception as e:
        print(f"Error verifying OTP: {e}")
        return False

Authentication

To use the QuantumOTP API, you need to authenticate your requests using an API key. Include your API key in the Authorization header of each request: 

Authorization: Bearer YOUR_API_KEY

API Endpoints

The QuantumOTP API provides two main endpoints for generating and verifying OTPs.

Generate OTP

Generate a new one-time password for a specific email address.

HTTP Request

POST https://api.quantumotp.com/generate

Request Body

{
  "email": "user@example.com",
  "length": 6,
  "expiry": 300
}

email (string, required): The email address to associate with the OTP.

length (integer, optional): Length of the OTP. Default is 6.

expiry (integer, optional): Expiry time in seconds. Default is 300 (5 minutes).

Response

{
  "success": true,
  "otp": "873421",
  "expires_in": 300
}

Verify OTP

Verify if a one-time password is valid for a specific email address.

HTTP Request

POST https://api.quantumotp.com/verify

Request Body

{
  "email": "user@example.com",
  "otp": "873421"
}

email (string, required): The email address associated with the OTP.

otp (string, required): The OTP to verify.

Response

{
  "success": true,
  "valid": true
}

Error Codes

Status Code Error Code Description
400 INVALID_REQUEST The request is missing required parameters or has invalid values.
401 UNAUTHORIZED The API key is missing or invalid.
404 NOT_FOUND The requested resource was not found.
429 RATE_LIMITED Too many requests. Please try again later.
500 SERVER_ERROR An internal server error occurred.
